# Titan Concrete Solutions — Website

A premium, agency-style website for Titan Concrete Solutions.

Designed to reflect strength, precision, and modern industrial craftsmanship.

---

## 🔥 Overview

This is a cinematic, high-end construction company website featuring:

- Full-screen hero section
- Industrial typography
- Structured grid-based layouts
- Project gallery
- Services section
- About & credibility positioning
- Testimonials
- Contact form
- Responsive design
- Smooth scroll and micro-interactions

Built to feel like a $50,000+ creative agency production.

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Custom Properties / Grid / Flexbox)
- Vanilla JavaScript
- Intersection Observer API
- Responsive design (mobile optimized)

No frameworks. No templates. Fully custom layout.
